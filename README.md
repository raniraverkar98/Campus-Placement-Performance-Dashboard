# 🎓 Campus Placement Performance Dashboard

An interactive **Power BI** dashboard built to analyze and visualize campus placement data, helping institutions track student placement trends and performance metrics at a glance.

---

## 📊 Overview

This project provides a comprehensive view of campus placement performance through dynamic visuals and filters. It enables placement officers, faculty, and management to make data-driven decisions.

---

## ✨ Features

- 📈 **Placement Trends** – Year-wise and branch-wise placement rate analysis
- 🏢 **Company Insights** – Top recruiters, offer counts, and sector distribution
- 💰 **Salary Analysis** – Average, highest, and lowest CTC breakdowns
- 🎓 **Student Performance** – Correlation between academic scores and placement outcomes
- 🔍 **Interactive Filters** – Slice data by year, department, gender, and more

---


## 🛠️ Tools & Technologies

| Tool |

| Power BI Desktop | Dashboard development & visualization |
| Microsoft Excel / CSV | Data source |
| DAX | Custom measures & calculations |

## 🎯 Purpose

The purpose of this project is to provide educational institutions with a **data-driven tool** to monitor and evaluate campus placement activities. 
  It aims to:

- Help **placement cells** track recruitment progress efficiently
- Identify **top-performing branches** and areas needing improvement
- Assist **management** in making informed decisions to enhance placement strategies
- Provide **students & faculty** with transparent insights into placement trends and outcomes
